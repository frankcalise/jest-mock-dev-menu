# React Native Jest Reproducer
This project reproduces an error seen when mocking parts of `react-native` while using `requireActual` as well.

## Get started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the app

   ```bash
   pnpm run test
   ```

3. Observe the following error
